Frameworks Web

O NetBeans IDE suporta os seguintes frameworks comuns da Web:

  • Zend Framework (screencast)
  • Framework Symfony2
  • Framework Symfony1 (screencast)

Em geral, você pode criar novos projetos PHP com esses frameworks, executar comandos do framework, navegar do controlador (ação) para sua view, usar anotações do framework e autocompletar o código no editor, e personalizar suas configurações.

 

Documentando Código PHP com ApiGen

É possível usar o NetBeans PHP IDE para gerar uma documentação do código PHP com ApiGen. É necessário realizar o download do ApiGen e certificar-se de que as opções do IDE reconheçam a instalação do ApiGen e de que o IDE encontrou o caminho correto do script ApiGen. Após configurar tudo corretamente, tudo o que precisa fazer é clicar com o botão direto do mouse no nó do projeto e selecionar Gerar Documentação. O IDE gera os arquivos de documentação no local de sua escolha.

Modelos Smarty


O NetBeans IDE suporta o framework mais usado para ORM (Mapeamento Relacional de Objetos), Framework Doctrine2 . Você pode executar facilmente todos os comandos do Doctrine2 diretamente do IDE, consulte screencast para obter mais informações.


O NetBeans IDE suporta o Composer. O Composer é uma ferramenta para gerenciamento de dependência no PHP. Ele permite declarar as bibliotecas dependentes de que seu projeto precisa e as instalará no projeto para você.


O Smarty oferece funcionalidades do editor NetBeans IDE padrão, como sintaxe em cores, modelos de código, recurso de autocompletar código incluindo documentação de tags, correspondência de tags em bloco, dicas de erro de análise e assim por diante. Consulte screencast para obter mais informações.

Modelos de Twig


O IDE oferece um suporte para modelos de Twig (que são fortemente conectados com o Framework Symfony). Ele fornece um recurso de autocompletar código para todos os elementos do Twig. Esses elementos são Tags, Filtros, Função, Testes e Operadores. Todos os elementos também são documentados. É possível usar um suporte a modelos de código incorporados.
